# Image Picking Tool (Python 3.12)

A fast, keyboard-driven image viewer to shortlist large image collections with resume support.

---

## Features

- Recursive image scanning (JPEG / PNG)
- Ultra-fast image rendering (4K-friendly)
- Keyboard navigation
- Instant background copy
- Resume from last viewed image
- Minimal dependencies

---

## Usage

- Run program
- Select New operation or resume operation at console
- In Image Window, press:
    - → for next image.
    - ← for previous image.
    - ⏎ to select current image.

---

## Requirements

- Python 3.12+
- Python packages: pygame, Pillow
- Linux / macOS / Windows
